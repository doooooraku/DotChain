

# 0_プロダクト戦略.md（DotChain ）

## 1. プロダクトの概要

### 1.1 アプリ名

- **DotChain (ドットチェーン)**
- 読み方：**ドットチェーン**
- キャッチコピー：
  **「脳が喜ぶ、指先の快感。世界一ミニマルな習慣スイッチ。」**

### 1.2 コンセプト（一言でいうと？）

**「『義務感』を消し去る、デジタル・フィジェット（手遊び）習慣ツール」**

> **解説**：
> 多くのアプリは「管理すること」を目的にしますが、DotChainは**「ボタンを押した瞬間の気持ちよさ（ASMR的快感）」**を最優先にします。
> 「記録のために押す」のではなく、**「押したいから記録する」**という逆転の発想です。

### 1.3 ミッション（解決する課題）

- **ターゲットの悩み（Pain）**：
  - 「高機能なアプリは設定が面倒で、3日で飽きる」
  - 「『今日もできなかった』と自分を責めてしまい、自己肯定感が下がる」
  - 「ADHD気質で、報酬（ご褒美）がすぐに貰えないと続かない」
- **提供する価値（Gain）**：
  - **即時報酬**：押した瞬間に「音・光・振動」で脳に報酬を与える。
  - **罪悪感ゼロ**：途切れても責めない。再開すればすぐに美しいチェーンが復活する。
  - **世界共通**：言葉への依存を極限まで減らし、直感だけで使える。

## 2. ターゲットユーザー（ペルソナ）

### 2.1 メインターゲット

- **属性**：Z世代〜ミレニアル世代（10代後半〜30代）
- **地域**：**全世界（特に19言語対応エリア）**
- **特性**：
  - **ミニマリスト**：ごちゃごちゃした画面や広告を嫌う。
  - **感覚派**：説明書を読まず、直感で操作したい。
  - **ニューロダイバーシティ層**：ADHDや「飽きっぽい」自覚があり、ゲーミフィケーション（音や光）による刺激を求めている。

## 3. 市場分析と勝ち筋（Strategy）

### 3.1 競合との差別化（ポジショニング）

| アプリ名 | 特徴 | DotChain の勝ち筋 |
| :--- | :--- | :--- |

| **Habitica** | RPGゲーム風 | **「大人っぽさ」と「即効性」**。<br>レベル上げすら面倒な人に、0.1秒の快感を提供する。 |
| **スマホ標準** | 事務的・無料 | **「エモーショナル（感情）」**。<br>事務作業ではなく「儀式」にする。 |

### 3.2 独自の強み（UVP）

**"Sensory Over Strategy"（戦略より感覚）**

1.  **Haptic Design (触覚)**：
    物理ボタンを押したような、重厚でリッチな振動（`expo-haptics`）。
2.  **Visual Chain (視覚)**：
    GitHubの草（Contribution Graph）のように、自分の努力が「アート」になる。
3.  **Global Native (世界対応)**：
    最初から19言語・RTL（右から左レイアウト）に完全対応。翻訳アプリのような違和感を消す。

## 4. 機能要件（V1.0 完成主義）

**「後から機能を追加しない」**ため、初期リリースですべてを盛り込みます。

### 4.1 無料版 (Free) - 撒き餌

- **習慣登録**：3つまで。
- **機能**：記録、ヒートマップ表示、標準の振動・音。
- **広告**：画面下部バナーのみ（操作を邪魔しない）。
- **ウィジェット**：使用可能（これが継続の鍵）。

### 4.2 有料版 (Pro) - 収益源

- **習慣登録**：無制限。
- **カラーテーマ**：全色開放（ネオン、パステル、サイバーパンク等）。
- **アプリアイコン変更**：ホーム画面カスタマイズ用アイコン開放。
- **広告非表示**。

### 4.3 実装しない機能（やらないことリスト）

- **× アカウント登録**：個人情報を持たない。プライバシー最強。
- **× クラウド同期**：V1.0ではオフライン完結。機種変更時はOSのバックアップ機能に任せる。
- **× ソーシャル共有**：他人と比べない。「自分との戦い」に集中させる。
- **× 複雑なグラフ**：「先月比◯％」などの数字は見せない。ドットの数だけで語る。
